== Description ==

Use Indiewire Player Embed to embed a Video from [Indiewire.com](http://www.indiewire.com/ "") into any page/post on your site.

Embed the Indiewire Player using shortcode '[indiewire id="VIDEO-ID-HERE"]'; replacing 'VIDEO-ID-HERE' with the id of the desired Indiewire Video.

= Useful plugins: =
* ["iframe"](http://wordpress.org/extend/plugins/iframe/ "") - Use iframe to embed video from YouTube or Vimeo or to embed Google Map or just to embed content from external page with a shortcode.
* ["Embedded Video"](http://wordpress.org/extend/plugins/embedded-video-with-link/ "") - Easy embedding of videos from lots of portals or local video files of many file formats with optional generation of a corresponding link.

== Other Notes ==

= Indiewire Player Embed Parameters: =
* **id** - id of the video you wish to display '[indiewire id="37d77890-1175-11e2-b9e5-22000a1d0930"]'

== Installation ==

1. Install and activate the plugin on the Plugins page.
2. Add the shortcode '[indiewire id="VIDEO-ID-HERE"]' to page or post content, replacing 'VIDEO-ID-HERE' with the id of the desired Indiewire Video.
